- python3 vs python2
    - python3 is the latest version of python and has many new features and improvements over python2.
    - python2 is no longer supported and should not be used for new projects.

- Advantages of python over other languages
    - easy to learn and use
    - large standard library
    - cross-platform compatibility
    - strong community support
    - good for data science and machine learning

- disadvantages of python
    - slower than some other languages like C++ and Java
    - not suitable for mobile app development
    - can be memory intensive

- what is pip?
    - pip is a package manager for python that allows you to install and manage third-party libraries and packages.

- What is pep8?
    - pep8 is a style guide for python code that provides guidelines for writing clean and readable code.
    - it covers topics such as indentation, naming conventions, and code layout.
    - following pep8 can help improve the readability and maintainability of your code.
    - you can use tools like flake8 or pylint to check your code for pep8 compliance.
    - the guidelines include using 4 spaces for indentation, limiting lines to 79 characters, and using descriptive variable names.
    - following pep8 can also help you avoid common pitfalls and improve the overall quality of your code.
